## Quick checks first

1. Refresh once and retry.
2. Confirm your network is stable.
3. Retry from [Practice](/practice) with no filters, then re-apply filters.

## Common issues

### Problems fail to load

- Clear filter/search combinations and retry.
- Test base route first: `/practice`.
- If a deep link fails, open the pattern root then navigate to the problem detail page.

### Hint or AI response unavailable

- This can happen during temporary provider/network interruptions.
- Continue with manual plan/code/reflect flow and retry hint call.

### Session behavior feels stale

- Open a new problem session from [Practice](/practice).
- Check your latest activity in [Dashboard](/dashboard).

### Mobile layout feels cramped

- Use landscape for long coding/editing phases.
- Keep one active task at a time (plan, then code, then test).

## Support-ready bug report template

Include:

- page URL
- exact action taken
- visible error message
- browser + device
- timestamp
