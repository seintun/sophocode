## What this page helps you do

Use the Practice workspace to choose problems intentionally instead of randomly. Good interview prep is less about volume and more about sequence.

## How to pick the right problem

Use this order:

1. Pick a **pattern** first (`/practice?pattern=...`).
2. Choose **difficulty** based on your recent confidence.
3. Use **search** only when you want a specific drill.

If you're uncertain, start with one easy and one medium problem in the same pattern.

## Suggested daily flow

| Step | Action                                         | Time      |
| ---- | ---------------------------------------------- | --------- |
| 1    | Solve one warm-up in a familiar pattern        | 10-15 min |
| 2    | Solve one stretch problem in your weak pattern | 20-30 min |
| 3    | Reflect and note one mistake category          | 5 min     |

This gives you consistency without burnout.

## Common mistakes

- Jumping patterns too often (no repetition signal).
- Practicing only hard problems.
- Skipping reflection after accepted solutions.
