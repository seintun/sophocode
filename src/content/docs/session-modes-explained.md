## Why session mode matters

Different interview skills require different pressure levels. Mode selection changes how you practice, not just what you solve.

## The three modes

### Self Practice

Best for learning a new pattern. Lower pressure, focus on understanding.

### Coach Me

Best for structured improvement. Sophia nudges you through Clarify -> Plan -> Code -> Reflect.

### Mock Interview

Best for simulation. Use when you already know the pattern and want communication + composure practice.

## Which mode should you pick?

| If your goal is...                | Use this mode  |
| --------------------------------- | -------------- |
| Learn a new pattern               | Self Practice  |
| Fix recurring mistakes            | Coach Me       |
| Prepare for real interview timing | Mock Interview |

## Common mistake

Using Mock Interview too early. Simulation without fundamentals creates stress without retention.
