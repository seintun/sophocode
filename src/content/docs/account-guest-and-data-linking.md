## Guest mode in Sophocode

You can start immediately without sign-up. Guest sessions are designed for low-friction interview practice.

## What gets tracked

- session attempts
- hint usage
- mastery signals
- pattern-level progress

## When to create an account

Create an account when you want more persistent cross-session continuity and longer-term planning.

## Data linking expectations

If you start as guest and later authenticate, prior practice activity is linked where supported by your active session context.

## Privacy and control

- You can practice without uploading local code files.
- Session data is used to power feedback and progress views.

## FAQ

### Do I need an account to practice problems?

No. Account creation is optional for initial use.

### Why does my progress look different across browsers?

Guest identity is browser/session-context based; account-linked usage is recommended for continuity.
